# Burger Timer

Wrote it because I wanted to learn flutter

## Explanation

This Project is a Cooking timer, in the app bar you select the time in seconds it will take to cook a Medium of whatever you want to cook, every level of coocking adds 15 seconds (MW = M+15, W=M+30, WD=M+45)
